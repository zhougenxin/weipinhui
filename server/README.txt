项目名称：唯品会
需要链接互联网图片使用的是在线图片
环境：本地服务器
首页：index.html
还有商品列表页（buy.html），商品详情（foods.html），登录页面（login.html），注册页面（zuce.html），购物车页面（shop.html）


使用JQuery框架，Cookie插件，validtae插件等。


主页：
	    三级导航；
	    楼层floor；
	    轮播图：插件；
	    点击商品跳转商品列表

商品列表页面：
	   AJAX请求数据渲染页面；

商品详情页：
	    放大镜功能。
	    加入购物车按钮有效

购物车页面：
	  
	    购物车上的价格数量，全选数量价格完成，结算按钮无效
	  
    
注册页面：
     	调用接口，简单的注册，注册成功5秒跳转登录页。

登陆页面：
     	调用接口，简单的注册，注册成功1秒跳转登录页。
